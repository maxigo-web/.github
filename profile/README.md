# maxigo-bot

Go ecosystem for building bots on [Max Messenger](https://max.ru).

| Package | Description | Status |
|---------|-------------|--------|
| [maxigo-client](https://github.com/maxigo-web/maxigo-client) | Idiomatic HTTP client for Max Bot API | [![release](https://img.shields.io/github/v/release/maxigo-web/maxigo-client)](https://github.com/maxigo-bot/maxigo-client/releases) |
| [maxigo-bot](https://github.com/maxigo-web/maxigo-bot) | Bot framework with router, middleware, and context | [![release](https://img.shields.io/github/v/release/maxigo-bot/maxigo-bot)](https://github.com/maxigo-bot/maxigo-bot/releases) |
| maxigo-proxy | Telegram → Max proxy — run existing Telegram bots on Max | Planned |
